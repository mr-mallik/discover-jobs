# Discover Jobs

Discover Jobs is a Python-based web scraper designed to automate the process of finding job listings across multiple job boards. It uses Selenium for web scraping and smtplib for sending email notifications about new job postings that match specified criteria.

Currently supports scraping from the following job boards:
- [Indeed UK](https://www.uk.indeed.com/)
- [LinkedIn](https://www.linkedin.com/jobs/)
- [Jobs.ac.uk](https://www.jobs.ac.uk/)
- [Find a job dwp](https://findajob.dwp.gov.uk/)
- [Innovate UK KTP](https://iuk-ktp.org.uk/)
